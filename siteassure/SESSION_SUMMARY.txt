Implemented UI polish for the main SiteAssure workflow screens:

- Flags screen: improved card layout, statuses, empty states, and note-taking UX.
- Record screen: improved amendment preview and version-history presentation.
- Dashboard screen: refreshed summary cards and visual hierarchy.

- Implemented automatic provisioning via the existing starter-model download path when the model is missing.
- Hooked this logic into the Tauri transcription command so recording can recover instead of failing immediately.
- Added a regression test covering the model-availability path.
