    Run it: serve over a local server so the worker + model cache work, e.g. python3 -m http.server then open http://localhost:8000 in Chrome or Edge (best WebGPU support). The first run downloads the model once, then it works fully offline.

    Make it installable everywhere: add a manifest.json + service worker and any phone/desktop can "Add to Home Screen" / "Install app" as a PWA.
