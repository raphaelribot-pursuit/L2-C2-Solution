3. Criticisms (honest UX gaps)
A. Visual hierarchy — functional, not product-grade
The UI reads as a well-built developer tool, not a consumer notes app like Whisper Notes. Dark panels, system fonts, equal-weight buttons — everything competes at the same volume. Jimmy’s PRD wanted a focal drop zone; today it’s a one-line hint:

“Drop an audio or video file here, or use the buttons below.”

That’s instruction, not invitation. New users don’t get a clear “this is where you put audio.”

B. YouTube is P0 in README, P2 in the UI
README says YouTube URL import is P0. In the app, URL import lives inside Advanced options, behind a collapsed panel. That’s a product priority mismatch — your most differentiated flow is buried.

C. Advanced panel is doing too many jobs
It mixes:

One-time setup (model download, yt-dlp, models folder)
Per-session prefs (language, GPU)
Alternate input (URL import)
Users opening Advanced for language see model paths, download buttons, and yt-dlp — cognitive load Jimmy’s PRD was trying to reduce.

D. Vertical stacking doesn’t scale
On a 720px-tall window with Advanced open + library + transcript, users scroll a lot. Whisper Notes–style apps often use sidebar library + main transcript on wider layouts. You have 960px width but cap content at 720px — wasted horizontal space.

E. Transcript toolbar is crowded
Copy · Export TXT · Export SRT · Export VTT · Delete — five actions in one row. At 720px this will wrap awkwardly. Export formats belong in one Export ▾ menu.

F. Copy drift / stale strings
The onboarding panel still says ~150 MB (Medium only) while the app supports Large (~1.6 GB). Small trust erosion.

G. No design system = iteration tax
Every UX tweak touches a 1,700-line file and a 900-line stylesheet. That’s why EmptyStateHero keeps getting deferred — the cost of extracting it is high, but the cost of not extracting it is higher now.

H. About dialog is engineer-facing
release_artifact, Host CPU, GPU → CPU fallback — fine for you and Jimmy, heavy for “privacy-conscious student who wants notes transcribed.”
